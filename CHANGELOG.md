# Changelog

## 0.1.0

- Initial public package for Zhiyun MOLUS control experiments.
- USB CDC probing, status reads, registration, frame planning, and diagnostics.
- BLE discovery, endpoint inspection, endpoint testing, and direct/worker transport paths.
- Python SDK surfaces for direct lights, controllers, integrations, projects, and named rigs.
- HTTP, OSC, Art-Net, and sACN integration adapters for media-production workflows.
- Scene presets, cues, serialized plans, setup profiles, readiness gates, and requested-state event streams.
