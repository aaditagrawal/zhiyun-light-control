# Documentation Index

Start here if you are deciding which layer of the project to use.

## Task Guides

- [Getting started](getting-started.md): install, check hardware, validate a
  light, and understand alpha control status.
- [CLI](cli.md): `zlight` commands for probing, validation, direct control,
  dry-run plans, metadata, and USB locking.
- [Python SDK](python-sdk.md): controllers, integration setup reports, setup
  profiles, primitive gates, and no-I/O planning helpers.
- [HTTP bridge](bridge-http.md): local JSON API, readiness, state/events,
  planning, execution, CORS, and bridge client helpers.
- [Media bridges](media-bridges.md): OSC, Art-Net, and sACN server behavior.
- [BLE](ble.md): service profiles, macOS helper flow, endpoint inspection,
  endpoint confirmation, and worker isolation.
- [Rigs and projects](rigs-projects.md): multi-fixture rig JSON, project files,
  setup-profile bundles, group helpers, and plan execution.
- [Hardware notes](hardware-notes.md): live G60 field notes, object-control
  discovery results, USB descriptors, and BLE observations.
